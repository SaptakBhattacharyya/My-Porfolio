# Saptak Bhattacharyya Portfolio

A polished personal portfolio built with React and Vite showcasing my resume, projects, skills, certificates, and education.

##  Project Overview

This portfolio is a modern, interactive personal site for:
- Full-stack development experience
- Resume summary and achievements
- Project showcases with live demos
- Skill cloud and technical categories
- Certificates and education timeline
- Contact form and social links
- Theme switcher (dark/light mode)

##  Core Features

- Hero section with a personal summary and call-to-action buttons
- About section with animated text and developer narrative
- Skills section with a 3D TagCloud and categorical skill cards
- Projects gallery with card layout and demo/source links
- Certificate showcase with lightbox preview
- Education timeline with academic details and SGPA
- Complete Resume section (printable/PDF option)
- Contact section with quick contact info and submission form
- Smooth scrolling using Lenis and UI animations using Framer Motion

##  Component Structure

- `src/App.jsx` - Main app structure and global theme state
- `src/components/Navbar.jsx` - Responsive nav + theme toggle
- `src/components/About.jsx` - About me content
- `src/components/Skills.jsx` - Skill cloud and skills grid
- `src/components/Projects.jsx` - Project cards and links
- `src/components/Certificates.jsx` - Certificates and modal preview
- `src/components/Education.jsx` - Education timeline
- `src/components/Resume.jsx` - Printable resume layout with education, skills, projects, and achievements
- `src/components/Contact.jsx` - Contact form and social profiles

## 🛠️ Tech Stack

- React
- Vite
- Framer Motion
- Lenis (smooth scrolling)
- TagCloud
- Lucide Icons
- CSS modules (component-scoped default CSS files)
